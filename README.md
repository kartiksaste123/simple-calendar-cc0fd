# Simple Calendar

A beautiful and responsive calendar application built with Next.js 14 and Tailwind CSS.

## Features

- 📅 Clean and modern calendar interface
- 🌓 Dark/Light mode toggle
- 📱 Fully responsive design
- ✨ Smooth animations and transitions
- 🎯 Date selection functionality
- 🔍 Month navigation
- 🎨 Beautiful gradient backgrounds
- 📍 Today's date highlighting
- 🎪 Random event indicators (demo)

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **React Hooks** - For state management

## Customization

The calendar is highly customizable through Tailwind CSS classes. You can modify:

- Color schemes in `tailwind.config.js`
- Layout and spacing in the main component
- Animation durations and effects
- Dark mode styles

## Building for Production

```bash
npm run build
npm start
```
